SQL LEN() Function

The LEN() function returns the length of a string in bytes.

Note: Depending on the SQL language, LEN() may also be called LENGTH().

Syntax

LEN(string)

Parameter Values

  • string: Required. The string to calculate the length for.

LEN() Example

The following SQL statement shows each patient's first_name and the length of that name: 

SELECT first_name, LEN(first_name) AS length_of_name
FROM patients;
