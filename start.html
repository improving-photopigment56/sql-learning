Bienvenido a SQL Cheatsheet

SQL (Structured Query Language) es un lenguaje estándar utilizado para interactuar con bases de datos relacionales. Permite crear, modificar y consultar datos de manera eficiente. Esta cheatsheet está diseñada para ayudarte a aprender y recordar las instrucciones más comunes de SQL.

Qué aprenderás aquí:

  • Query Basics: SELECT, INSERT, UPDATE, DELETE y WHERE.
  • Filtrado y orden: Operadores, ORDER BY, LIKE, IN, BETWEEN, GROUP BY y HAVING.
  • Funciones: Funciones agregadas (COUNT, SUM, AVG, MAX, MIN), de cadena, numéricas y de fecha.
  • Tablas y Constraints: CREATE, ALTER, DROP TABLE, PRIMARY KEY, FOREIGN KEY, UNIQUE, CHECK, DEFAULT.
  • Índices y Auto Increment: Mejora el rendimiento de las consultas con índices y claves autoincrementales.
