SQL CREATE TABLE Statement

The CREATE TABLE statement is used to create a new table in a database.

CREATE TABLE Syntax

CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    column3 datatype,
    ...
);

The column parameters specify the names of the columns, and the datatype parameter specifies the type of data the column can hold (e.g., varchar, integer, date).

SQL CREATE TABLE Example

Create a table called Persons with 5 columns:

CREATE TABLE Persons (
    PersonID int,
    LastName varchar(255),
    FirstName varchar(255),
    Address varchar(255),
    City varchar(255)
);

Create Table Using Another Table

You can create a new table as a copy of an existing table. All or specific columns can be selected:

CREATE TABLE new_table_name AS
SELECT column1, column2,...
FROM existing_table_name
WHERE ...;

Example: Copy selected columns from the patients table:

CREATE TABLE TestPatients AS
SELECT patient_id, first_name, last_name
FROM patients;

-- Verify the new table
SELECT * FROM TestPatients;
