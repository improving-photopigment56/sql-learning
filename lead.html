SQL LEAD() Window Function

The LEAD() function returns the record offset by the specified amount forward.

Syntax

LEAD(expression [, offset])

Parameter Values

  • expression: Required. The value to be returned based on the specified offset. Must be a scalar value; cannot be an analytic function.
  • offset: Optional. Number of rows forward from the current row. Default is 1. Must be positive; cannot be negative or an analytic function.

LEAD() Example

The following SQL statement displays each patient's first_name along with the first_name of the next patient:

SELECT
  patient_id,
  first_name,
  LEAD(first_name, 1) OVER() AS next_name
FROM patients;
