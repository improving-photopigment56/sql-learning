SQL POWER() Function

The POWER() function returns the value of a number raised to the power of another number.

Note: Depending on the SQL language, POWER() may also be called POW().

Syntax

POWER(base, exponent)

Parameter Values

  • base: Required. The base number.
  • exponent: Required. The exponent (power to raise the base).

POWER() Example

The following SQL statement returns 8 raised to the power of 3:

SELECT POWER(8, 3);
