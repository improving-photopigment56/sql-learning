SQL HAVING Clause

The HAVING clause was added to SQL because the WHERE keyword cannot be used with aggregate functions.

HAVING Syntax

SELECT column_name(s)
FROM table_name
WHERE condition
GROUP BY column_name(s)
HAVING condition
ORDER BY column_name(s);

SQL HAVING Examples

The following SQL statement lists the names of patients with a common first_name (> 30 occurrences):

SELECT COUNT(*), first_name
FROM patients
GROUP BY first_name
HAVING COUNT(*) > 30;

The following SQL statement lists the names of patients with a common first_name (> 30 occurrences), sorted high to low:

SELECT COUNT(*), first_name
FROM patients
GROUP BY first_name
HAVING COUNT(*) > 30
ORDER BY COUNT(*) DESC;
