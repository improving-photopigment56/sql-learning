SQL CONCAT() Function

The CONCAT() function adds two or more expressions together.

Note: Depending on the SQL language, there may be built-in syntax for concatenation such as + or ||.

Syntax

CONCAT(expression1, expression2, expression3, ...)

Parameter Values

  • expression1, expression2, expression3, etc.: Required. The expressions to add together. If any expression is NULL, the result will be NULL.

CONCAT() Example

The following SQL statement concatenates the first_name and last_name of each patient:

SELECT CONCAT(first_name,' ', last_name) AS full_name
FROM patients;
