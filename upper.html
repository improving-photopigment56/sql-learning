SQL UPPER() Function

The UPPER() function returns a string in uppercase. Example: 'ExAmple' → 'EXAMPLE'.

Note: Depending on the SQL language, UPPER() may also be called UCASE().

Syntax

UPPER(string)

Parameter Values

  • string: Required. The string to convert to uppercase.

UPPER() Example

The following SQL statement shows each patient's first_name in uppercase:

SELECT first_name, UPPER(first_name) AS uppercase_name
FROM patients;
