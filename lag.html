SQL LAG() Window Function

The LAG() function returns the record offset by the specified amount.

Syntax

LAG(expression [, offset])

Parameter Values

  • expression: Required. The value to be returned based on the specified offset. Must be a scalar value; cannot be an analytic function.
  • offset: Optional. Number of rows back from the current row. Default is 1. Must be positive; cannot be negative or an analytic function.

LAG() Example

The following SQL statement displays each patient's first_name along with the first_name of the previous patient:

SELECT
  patient_id,
  first_name,
  LAG(first_name, 1) OVER() AS previous_name
FROM patients;
