SQL RAND() Function

The RAND() function returns a random number between 0 and 1.

Note: Depending on the SQL language, RAND() may also be called RANDOM().

Note: This implementation does not support the seed parameter.

Syntax

RAND(seed)

Parameter Values

  • seed: Optional. If specified, it returns a repeatable sequence of random numbers. If not specified, it returns a completely random number.

RAND() Example

The following SQL statement displays a random number between 0 and 1:

SELECT RAND();
