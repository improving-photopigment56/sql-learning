SQL LOWER() Function

The LOWER() function returns a string in lowercase. Example: 'ExAmple' → 'example'.

Note: Depending on the SQL language, LOWER() may also be called LCASE().

Syntax

LOWER(string)

Parameter Values

  • string: Required. The string to convert to lowercase.

LOWER() Example

The following SQL statement shows each patient's first_name in lowercase:

SELECT first_name, LOWER(first_name) AS lowercase_name
FROM patients;
