SQL IFNULL(), ISNULL(), COALESCE(), and NVL() Functions

The IFNULL function goes by many names depending on what SQL dialect you are using. We use IFNULL but your specific language may use ISNULL, COALESCE, or NVL — most function the same.

Example Table

SELECT first_name, allergies FROM patients;

The allergies column displays NULL for some rows. We want it to display a default value instead.

Solutions

MySQL

The IFNULL() function returns an alternative value if an expression is NULL:

SELECT first_name, IFNULL(allergies,'none') AS allergies
FROM patients;

Or using COALESCE():

SELECT first_name, COALESCE(allergies,'none') AS allergies
FROM patients;

SQL Server

The ISNULL() function returns an alternative value if the expression is NULL:

SELECT first_name, ISNULL(allergies,'none') AS allergies
FROM patients;

MS Access

The IsNull() function returns TRUE (1) if the expression is NULL, otherwise FALSE (0):

SELECT first_name, IIF(ISNULL(allergies,0),allergies) AS allergies
FROM patients;

Oracle

The NVL() function achieves the same result:

SELECT first_name, NVL(allergies,'none') AS allergies
FROM patients;
