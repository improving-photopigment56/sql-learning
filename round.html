SQL ROUND() Function

The ROUND() function returns a number rounded to the nearest specified decimal position.

Syntax

ROUND(number, decimals)

Parameter Values

  • number: Required. The numeric value to round.
  • decimals: Optional. The number of decimal places to round to. If omitted, the result will be an integer (no decimals).

ROUND() Example

The following SQL statement rounds 135.375 to 2 decimal places:

SELECT ROUND(135.375, 2);
